# MindCare Backend API

Mental Health Support Platform Backend built with Node.js, Express, and MongoDB.

## Features
- User authentication & authorization
- Mood tracking with statistics
- Support forum with moderation
- Therapist directory & appointment booking
- Real-time notifications
- File uploads
- Email notifications
- Rate limiting & security

## Prerequisites
- Node.js (>= 16.0.0)
- MongoDB (>= 4.4)
- Redis (optional, for rate limiting)
